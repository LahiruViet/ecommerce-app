# e-commerce-app  

- This is a spring boot project for microservice architecture.
- available services are
  - config server
  - eureka discovery server
  - api gateway
  - product service
  - order service
  - customer service
  - notification service
  - payment service
 
## Tech Stack  

- Java 21
- Spring Boot 3
- Mongo DB
- postgres
- Kafka
- Keycloak
- Docker
- zipkin
